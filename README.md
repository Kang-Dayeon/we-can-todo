# we-can-todo

## Stacks
* Environment
  - Visual Studio Code
  - github
  
* config
  - NPM
  
* Development
  - React
  - React Router
  - Redux
  - Axios
  - Styled component
  - Typescript
  - Express
  - MySQL

## Requirements

* Node.js 20.9.0
* Npm 9.7.1
* React 18.2.0
* React-router-dom 18.2.0
* Typescript 5.0.4
* Axios 1.4.0
* Redux 4.2.1
* Express 4.18.2

## Installation
<pre>
<code>// ** start
// cd client
$ npm run build

  
// cd server
$ npm run server
</pre>
